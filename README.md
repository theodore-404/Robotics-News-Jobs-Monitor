# Robotics-News-Jobs-Monitor

Repository Mini Task AI and Automation with n8n MySkill

Robotics News & Jobs Monitor

This is an automated workflow built with n8n that collects the latest robotics news and robotics job opportunities from RSS feeds, summarizes them using Google Gemini AI, and delivers a clean daily newsletter via Gmail.

Features
- Collects the latest robotics news from trusted RSS feeds.
- Collects robotics-related job opportunities.
- Uses Google Gemini AI to summarize important news.
- Sends a daily email newsletter automatically.
- Supports scheduled execution using n8n Schedule Trigger.

Tech Stack
- n8n
- Google Gemini API
- Gmail API
- RSS Feeds
- Javascript (Code Node)

Email Output
Every morning, the workflow generates a newsletter containing:
- Top Robotics News
- Robotics Job Opportunities
- AI-generated summaries
- Source links
